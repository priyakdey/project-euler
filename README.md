# Code for [Project Euler](https://projecteuler.net/)

One fine midnight, I decided to solve the Project Euler Problems, and learn some C.


### Description

This is a online challenge which helps students learn and be better at maths,
thinking and programming.


### Usage

- Util to create source files with pre-defined template.

```bash
$ ./cf.py <filename>    # example ./cf.py 1 - creates 1.c in root
```

- Run the source file.

```bash
$ make TC=<filename>    # example make TC=4 runs 4.c
```
